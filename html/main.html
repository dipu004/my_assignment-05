Question-2: What is the differetn between innerHTML ,innerText , and TextContent?

TextContent is all text contained by an element and all its children that are for formatting purposes only. innerText returns all text contained by an element and all its Child elements. innerHTML returns all text, icluding html tags . that is contained by an element.

Question-3: What is event delegation in DOM?

Event delegation is a technique in javaScript where a single event is attached to a parent element of attaching event listeners to multiple child elements.
